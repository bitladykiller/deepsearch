# 07 - 记忆系统

## 1. 概述

记忆系统是 Deep Research 的跨会话知识管理核心，使系统能够"记住"用户的偏好、历史交互和事实知识。

### 1.1 记忆系统架构

### 1.2 四种记忆类型

| 类型 | 枚举值 | 说明 | 存储后端 |
|------|--------|------|----------|
| 短期记忆 | `SHORT_TERM` | 当前会话上下文，消息历史 | Redis / MySQL / 内存 |
| 语义记忆 | `SEMANTIC` | 用户画像、事实知识、偏好 | MySQL + Milvus |
| 情景记忆 | `EPISODIC` | 历史任务、执行轨迹、对话记录 | MySQL + Milvus |
| 程序记忆 | `PROCEDURAL` | 系统提示、行为模式（已定义，未启用） | - |

```python
class BaseMemory(ABC):
```

## 3. 短期记忆（short_term.py）

### 3.1 ConversationBuffer 对话缓冲区

```python
class ConversationBuffer:
    def __init__(self, max_messages=20, max_tokens=4000, summary_threshold=10):
        self.max_messages = max_messages
        self.max_tokens = max_tokens
        self.summary_threshold = summary_threshold
        self.messages: List[BaseMessage] = []
        self.summary: Optional[str] = None
        self.token_count: int = 0
```

**消息压缩策略**：

```
消息历史：[msg1, msg2, ..., msg25, msg26, msg27, msg28]
                                   ↑
                        summary_threshold = 10

压缩后：
  summary = "msg1~msg18 的摘要"
  messages = [msg19, msg20, ..., msg28]
```

当 `len(messages) > max_messages` 时：
1. 将前 `len - summary_threshold` 条消息生成摘要
2. 保留最后 `summary_threshold` 条消息
3. 新摘要追加到已有摘要之后

### 3.2 ShortTermMemory 类

```python
class ShortTermMemory(BaseMemory):
    def __init__(self, ttl_seconds=3600, max_threads=100):
        self.ttl_seconds = ttl_seconds
        self.max_threads = max_threads
        # 存储结构: {thread_id: {"buffer": ConversationBuffer, "metadata": {}, "last_access": datetime}}
        self._storage: Dict[str, Dict[str, Any]] = {}
```

**核心特性**：
- TTL 过期：超过 `ttl_seconds` 未访问的线程自动清理
- 最大线程数限制：超过 `max_threads` 时清理最旧的线程
- 内存存储：进程退出即丢失

## 4. 长期记忆（long_term.py）

### 4.1 BaseLongTermMemory 基类

```python
class BaseLongTermMemory(BaseMemory, ABC):
    def _generate_embedding(self, text: str) -> List[float]:
        """生成文本的向量嵌入（简化版本）"""
        # 使用 MD5 哈希模拟，实际应使用 Embedding 模型
        hash_obj = hashlib.md5(text.encode())
        hash_bytes = hash_obj.digest()
        
        # 生成 384 维的伪向量
        embedding = []
        for i in range(0, len(hash_bytes), 2):
            val = (hash_bytes[i] + hash_bytes[i+1]) / 255.0
            embedding.append(val)
        
        # 扩展到 384 维
        while len(embedding) < 384:
            embedding.extend(embedding[:384 - len(embedding)])
        
        return embedding[:384]
    
    def _calculate_similarity(self, vec1, vec2) -> float:
        """计算余弦相似度"""
        dot_product = sum(a * b for a, b in zip(vec1, vec2))
        norm1 = sum(a * a for a in vec1) ** 0.5
        norm2 = sum(b * b for b in vec2) ** 0.5
        return dot_product / (norm1 * norm2) if norm1 and norm2 else 0.0
```

```python
def search(self, query, user_id=None, namespace=None, limit=5, **kwargs):
    query_embedding = self._generate_embedding(query)
    
    # 1. 从数据库获取所有匹配的记录
    rows = conn.execute(f"SELECT * FROM memories WHERE {where_clause}", params)
    
    # 2. 计算相似度并排序
    entries_with_scores = []
    for row in rows:
        entry = self._row_to_entry(row)
        
        # 向量相似度（权重 0.7）
        vec_score = self._calculate_similarity(entry.embedding, query_embedding)
        
        # 关键词匹配（权重 0.3）
        text_score = 0.5 if query.lower() in str(entry.content).lower() else 0.0
        
        # 综合分数
        total_score = vec_score * 0.7 + text_score * 0.3
        entries_with_scores.append((entry, total_score))
    
    # 3. 排序返回
    entries_with_scores.sort(key=lambda x: x[1], reverse=True)
    return [entry for entry, _ in entries_with_scores[:limit]]
```

### 5.4 会话摘要压缩

#### Redis 摘要压缩

```python
def _compress_redis_thread(self, tenant_id, user_id, thread_id):
    # 1. 获取所有消息
    raw_messages = self._redis_client.lrange(key, 0, -1)
    
    # 2. 如果消息数 <= max_messages，不需要压缩
    if len(raw_messages) <= self.short_term_max_messages:
        return
    
    # 3. 分割：待摘要 + 保留
    parsed = [json.loads(item) for item in raw_messages]
    split_at = len(parsed) - self.short_term_summary_threshold
    to_summarize = parsed[:split_at]
    keep_messages = parsed[split_at:]
    
    # 4. 生成新摘要
    existing_summary = self._redis_client.get(summary_key) or ""
    new_summary = self._summarize_text(existing_summary, to_summarize)
    
    # 5. 更新 Redis
    pipe = self._redis_client.pipeline()
    pipe.delete(key)
    pipe.rpush(key, *[json.dumps(item) for item in keep_messages])
    pipe.set(summary_key, new_summary, ex=self.short_term_ttl)
    pipe.expire(key, self.short_term_ttl)
    pipe.execute()
```

#### LLM 摘要生成

```python
def _summarize_text(self, existing_summary, history_slice):
    # 构造历史文本
    lines = [f"{item['role']}: {item['content']}" for item in history_slice]
    history_text = "\n".join(lines)
    
    if self._summary_llm is None:
        # 降级：简单截断
        combined = f"{existing_summary}\n{history_text}".strip()
        return combined[-4000:]
    
    # 使用 LLM 生成摘要
    prompt = (
        "你是对话压缩引擎。请在保留事实、偏好、结论、待办和约束的前提下进行递归摘要。\n"
        f"已有摘要：{existing_summary or '无'}\n"
        "新增历史：\n"
        f"{history_text}\n"
        "输出要求：100-300字，中文，结构紧凑。"
    )
    response = self._summary_llm.invoke([HumanMessage(content=prompt)])
    return str(response.content).strip()
```

```python
def persist_turn(self, tenant_id, user_id, thread_id, query, answer):
```

### 5.8 多租户隔离

所有记忆操作都携带 `tenant_id`，确保不同租户的数据完全隔离：

```
tenant_id + user_id + thread_id  →  唯一的记忆空间
```

- MySQL 查询：`WHERE tenant_id = %s AND user_id = %s`
- Milvus 过滤：`metadata["tenant_id"] == tenant_id AND metadata["user_id"] == user_id`
- Redis Key：`ma:short:{tenant_id}:{user_id}:{thread_id}`

## 6. 记忆工具函数（utils.py）

### 6.1 extract_memory_from_messages

从消息列表中提取事实和偏好：

```python
def extract_memory_from_messages(messages):
    facts = []
    preferences = []
    
    for msg in messages:
        content = str(msg.content).lower()
        
        # 提取事实（"我是XX"、"我叫XX"）
        if "我是" in content or "我叫" in content:
            facts.append(str(msg.content))
        
        # 提取偏好（"我喜欢"、"我不喜欢"）
        if "我喜欢" in content or "我不喜欢" in content:
            preferences.append(str(msg.content))
    
    return {"facts": facts, "preferences": preferences}
```

### 6.2 format_memories_for_prompt

将记忆条目格式化为 Prompt 注入文本：

```python
def format_memories_for_prompt(entries, max_length=1800):
    lines = ["## 相关记忆"]
    total = 0
    
    for entry in entries:
        snippet = str(entry.content)
        if len(snippet) > 120:
            snippet = snippet[:120] + "..."
        
        line = f"- [{entry.memory_type.value}] {snippet} ({entry.created_at.strftime('%Y-%m-%d')})"
        
        if total + len(line) > max_length:
            break
        
        lines.append(line)
        total += len(line)
    
    return "\n".join(lines)
```

### 6.3 merge_user_profile

合并用户画像：

```python
def merge_user_profile(existing, new):
    if existing is None:
        return new_data.copy()  # 源码参数名为 new_data
    merged = existing.copy()
    for key, value in new_data.items():
        if key in merged and isinstance(merged[key], dict) and isinstance(value, dict):
            merged[key].update(value)          # 字典递归合并
        elif key in merged and isinstance(merged[key], list) and isinstance(value, list):
            merged[key] = list(set(merged[key] + value))  # 列表去重合并
        else:
            merged[key] = value                 # 标量覆盖
    merged["_last_updated"] = datetime.now().isoformat()
    return merged
```
